# PropFlow

PropFlow is an AI-powered autonomous property management platform focused on automation, prediction, and optimization for the Indian rental ecosystem.

## Monorepo Layout

- `apps/api-server` - NestJS backend
- `apps/web-dashboard` - Next.js admin dashboard
- `apps/mobile-app` - Flutter tenant application
- `apps/ai-engine` - FastAPI AI engine
- `packages/ui` - shared UI package placeholder
- `packages/config` - shared config package placeholder
- `packages/utils` - shared utilities package placeholder
- `infra` - Docker, Nginx, Terraform, and scripts
- `docs` - project documentation

## Status

This repository currently contains the production-ready folder structure and starter placeholder files for the finalized PropFlow architecture.
